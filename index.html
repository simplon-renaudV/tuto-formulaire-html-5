Les formulaires en HTML 5

Qu'est-ce qu'un formulaire ? C'est la base pour des échanges entre le visiteur et le site web.

Javascript associé aux formulaires permet :

  • d'assister et de guider le visiteur
  • de contrôler la saisie
  • de faire des traitements (calcul, manipulation, de caractères)
  • d'envoyer les résultats au serveur

Il ne communique pas avec les bases de données

un formulaire doit être encadré par des balises <form> </form>

Code HTML 5 du formulaire


	<form id="formulaire" name="formulaire" method="POST">
	<!-- Il existe 2 méthodes pour envoyer un formulaire : GET et POST. C'est POST qui est le plus souvent utilisé -->	
			
		<input type="text" id="nom" name="nom" placeholder="Entrez votre nom" required>nom</input><br>
		<!-- le type text est un champ de texte simple. Placeholder est le texte qui apparait en grisé dans le champ. Required est un attribut qui signifie que l'utilisateur doit obligatoirement remplir ce champ -->

		<input type="email" id="mail" name="mail" placeholder="Entrez votre mail">email</input><br>
		<!-- le type email demande à l'utilisateur de rentrer une addresse email valide -->

		<input type="url" id="site" name="site" placeholder="Entrez l'addresse de votre site">site web</input><br>
		<!-- url, vous vous en doutez, non ? -->

		<input type="password" id="password" name="password" placeholder="Entrez un mot de passe" onchange="controle()">mot de passe (8 
		caractères minimum)</input><br>
		<!-- pour le mot de passe, les caractères sont remplacés par des points. Le onchange appelle une fonction javascript qui s'exécute lorsque l'utilisateur modifie le contenu de ce champ-->

		<input type="checkbox" id="check" name="check">J'accepte les termes du contrat utilisateur</input><br>
		<!-- des cases à cocher, plusieurs peuvent être cochées simultanément -->

		<input type="radio" id="radio1" name='radio' checked>Bouton radio 1</input><br>
		<input type="radio" id='radio2' name='radio'>Bouton radio 2 </input><br>
		<!-- des boutons radio, un seul peut être coché. Il faut tous qu'ils aient le même attribut name. Checked signifie qu'il est coché -->

		<input type="submit" id="bouton" name="bouton" value="Valider l'inscription"></input><br>
		<!-- le bouton pour envoyer le formulaire -->	

	</form>
